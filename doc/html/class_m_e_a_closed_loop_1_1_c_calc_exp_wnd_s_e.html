MEAClosedLoop.CCalcExpWndSE.CCalcExpWndSE ( int  tau)

Create SE calculator. Exponential multiplier decays to 1/e in tau samples, to 0.05 in 3*tau. Multiplier to be used in exponential sum, m = (tau-1)/tau. Consider 3*tau as a width of an exponential window.

Аргументы
pNumber of samples to decay weight function to 1/e
